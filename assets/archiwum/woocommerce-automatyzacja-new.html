FAQ

Najczęstsze pytania
o automatyzację WooCommerce

Czy mój sklep musi mieć jakieś specjalne wtyczki? +
Nie. Korzystam z natywnego REST API WooCommerce, które jest dostępne w każdej standardowej instalacji WordPress + WooCommerce. Nie musisz nic instalować przed naszą rozmową.
Czy automatyzacje działają gdy zmienię motyw lub dodam wtyczkę? +
Tak — automatyzacje komunikują się z WooCommerce przez API, nie przez interfejs graficzny. Zmiany w wyglądzie sklepu nie wpływają na działanie workflow. Aktualizacje wtyczek mogą czasem wymagać drobnych korekt — to obejmuje gwarancja.
Czy moje dane klientów są bezpieczne? +
Tak. Podpisujemy NDA przed projektem. Dostępy (klucze API, hasła) przekazywane są przez szyfrowane kanały i usuwane po zakończeniu projektu. Dane klientów Twojego sklepu nigdy nie są przechowywane poza Twoimi systemami.
Czy n8n / Make.com to dodatkowy koszt dla mnie? +
n8n można hostować samodzielnie (praktycznie za darmo na VPS za ~20 zł/mies.) lub używać wersji cloud. Make.com ma darmowy plan dla mniejszej liczby automatyzacji. Doradzę najlepsze rozwiązanie pod Twój wolumen zamówień i budżet.
Czy mogę zamawiać przez Useme / bez faktury VAT? +
Tak, rozliczam się przez platformę Useme co jest wygodne dla obu stron. Możliwe też wystawienie rachunku lub faktury VAT w zależności od potrzeb.
Co jeśli WooCommerce zmieni API po wdrożeniu? +
WooCommerce zachowuje kompatybilność wsteczną API przez wiele lat. W przypadku złamania automatyzacji przez aktualizację WooCommerce — naprawiam bezpłatnie w ramach gwarancji, a po jej zakończeniu w ramach retainera lub za małą opłatą.
